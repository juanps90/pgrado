272 # Publica en el topico "topic_command" un String que indica el comando a ser ejecutado en el sistema.
273 # - INIT_LEARNING: Comando que indica el comienzo de una demostracion.
274 # - END_LEARNING: Comando que indica el fin de una demostracion. A este comando se le concatenar un "|" y un string que indica
275 # el nombre con el cual se persiste lo enseñado. Ejemplo: END_LEARNING|myDemo.xml.
276 # - PLAY: Comando que indica el inicio de una reporduccion. A este comando se le concatena un "|" y un string que indica
277 # la tarea a ser reporduciada.
278 # - STOP: Comando que indica el fin de una reproduccion en curso.
279 # - BAD: Comando que indica que la parte que se esta reproduciendo esta mal y por tanto debe ser eliminada.
280 # Este comando me premite eliminar un nodo de la red de comportamientos.
281 # - COME: Durante el aprendizaje el robot puede perder pasos relevantes. Este comando permite entrar en modo de aprendizaje
282 # durante la reproduccion e indicar asi los pasos faltantes. Esto nos permite agregar una subred a la red actual.
283 # - GO: Este comando perite retomar la reproduccion luego de haber entrado en modo aprendizaje con el comando COME
284 # - HERE: ------------------------
285 # - EXIT: Salir del sistema.
286 # Ademas de estos comandos tenemos otros comandos dedicados a calibrar colores.
