def inputs.processHeadVisionSensorData ( )

Retorna un String que representa los valores del headVisionSensor.

Devuelve
Devuelve un String que tiene el formato id sensor|numero de datos|dato 1|dato 2|etc. Numero de datos en este caso es 2. En dato 1 se guarda un valor entre 0 y 1. 0 indica que el objeto esta lo mas a la izquierda posible de la imagen. 1 indica que el objeto esta lo mas a la derecha posible de la imagen. En dato 2 se guarda un valor que indica el color del objeto segun las constantes establecidas en Const.
